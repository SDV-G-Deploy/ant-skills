# Русский мини-словарь ANT

Цель — не запретить английские термины, а объяснять их человеческим языком.

| Термин | Как объяснять проще |
|---|---|
| API | Способ, которым одна программа просит что-то у другой программы. |
| Endpoint | Конкретный адрес или функция, куда приложение отправляет запрос или откуда получает ответ. |
| Deploy / деплой | Публикация приложения или сайта, чтобы им могли пользоваться другие люди. |
| Build / билд | Подготовка приложения к запуску или публикации. |
| Dependency | Библиотека, которая нужна проекту для работы. |
| Package | Готовый кусок кода, который проект подключает как инструмент. |
| Repository / repo | Папка проекта вместе с историей изменений, обычно на GitHub. |
| Branch / ветка | Отдельная линия изменений, чтобы не ломать основную версию. |
| Commit | Сохранённый шаг в истории изменений. |
| Pull request / PR | Заявка на внесение изменений в основную версию проекта. |
| Merge | Объединение изменений из одной ветки с другой. |
| Environment variable | Скрытая настройка проекта, часто для ключей, паролей или адресов сервисов. |
| Token | Цифровой ключ/пропуск, который подтверждает доступ. |
| Schema | Правила того, как должны выглядеть данные. |
| Migration | Контролируемое изменение структуры базы данных. |
| Middleware | Промежуточный слой, который обрабатывает запрос до основной логики. |
| Runtime | Среда, в которой запускается код. |
| CI/CD | Автоматическая проверка и публикация проекта после изменений. |
| Production | Рабочая версия, которой пользуются реальные люди. |
| Staging | Тестовая версия, похожая на рабочую, но не для реальных пользователей. |
| Auth / authentication | Проверка, кто пользователь и имеет ли он доступ. |
| Authorization | Проверка, что пользователю разрешено делать конкретное действие. |
| Cache | Временное сохранение данных, чтобы быстрее отвечать или меньше загружать сервисы. |
| Webhook | Автоматическое сообщение от одного сервиса другому, когда что-то произошло. |
| SDK | Набор готовых инструментов для работы с сервисом или платформой. |
| CLI | Управление программой через команды в терминале. |
