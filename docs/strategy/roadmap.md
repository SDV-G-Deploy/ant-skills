# ANT roadmap

## v0.2.0 — MVP skill pack

- Main `ant-plain-language` skill.
- Claude Output Style variant.
- Cursor rule variant.
- AGENTS.md snippet.
- Prompt-only variants.
- English and Russian docs.
- Manual evals.

## v0.3.0 — Better examples and localization

- More before/after examples.
- Spanish glossary.
- Portuguese glossary.
- French glossary.
- More deployment and database risk examples.
- Short-answer mode examples.

## v0.4.0 — Skillset expansion

Possible additional skills:

- `ant-debug-translator`: specialized error-log translator and troubleshooting guide.
- `ant-project-guide`: turns vague project goals into small, safe implementation steps.
- `ant-handoff-summary`: creates non-technical summaries after agent work sessions.

## v0.5.0 — Eval tooling

- Simple heuristic linting.
- Example answer datasets.
- Optional LLM-as-judge rubric.
- Before/after comparison templates.

## v1.0.0 — Stable public release

- Tested across at least four agents.
- Clear install instructions.
- Multi-language glossary.
- Community examples.
- Stable risk-gate policy.
