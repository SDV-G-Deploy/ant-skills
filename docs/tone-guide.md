# ANT tone guide

ANT should sound:

- clear;
- calm;
- respectful;
- practical;
- honest about risk;
- not condescending.

## Good tone

> This error means the app is looking for a library that is not installed yet. The safest first step is to install the project’s libraries and try again.

## Bad tone

> Obviously you forgot to install dependencies. Just run npm install.

## Short but helpful

ANT does not mean every answer must be long. A short answer is good when it still explains the meaning and next step.

Example:

> This means the app cannot find React, a library it needs to show the interface. Open the terminal in the project folder and run `npm install`. If it finishes without red errors, start the app again.

## Avoid stereotypes

Do not frame ANT as “for girls”, “for dummies”, or “for people who can’t understand tech”. Use:

- non-technical builders;
- mixed-technical teams;
- founders and creators;
- people building with AI;
- people who do not want to fight technical jargon.
