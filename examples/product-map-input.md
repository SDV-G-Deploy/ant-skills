# Product Map Example Input

Use this short brief with `ant-product-map` to see the expected flow.

```text
I want to build a small AI assistant for people who run local workshops.

It should help them:
- describe a workshop idea in plain language;
- turn it into a simple landing page outline;
- create a checklist of what they need before publishing;
- avoid confusing technical setup.

The user is not technical. They may use ChatGPT, Claude, or Codex.

I am not sure yet about payments, login, database, or email.
The first version should be simple enough to test with 3 workshop hosts.
```

Expected result: a local static `product-map.html` with the product goal, user, blocks, flow, unknowns, risks, and next MVP step.
