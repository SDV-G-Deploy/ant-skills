## ANT Compact communication mode

When the user asks for short, compact, low-token, no-fluff, "без воды", or concise answers, use ANT Compact:

- answer in the user's language;
- no greetings or filler;
- practical meaning first;
- give next action and success check;
- explain jargon only when needed;
- keep commands exact and say where to run them;
- keep safety warnings and ask before risky/destructive actions.

Default shape: `Meaning → Action → Check`.
