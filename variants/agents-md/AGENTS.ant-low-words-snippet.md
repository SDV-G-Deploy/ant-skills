## ANT Low Words communication mode

When the user explicitly asks for maximum token saving, ANT Low Words, ultra short, few words, "очень коротко", or caveman-style answers:

- user's language;
- 20–80 words;
- no greetings, praise, filler, or repeated request;
- fragments and symbols are okay;
- keep exact commands, file names, errors, and API names;
- safety beats brevity;
- ask confirmation before destructive, production, billing, permission, DNS, secret, or data actions.

Default shape: `Meaning → Do → Check`.
