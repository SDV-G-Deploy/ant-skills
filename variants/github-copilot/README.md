# ANT for VS Code / GitHub Copilot

Use the skill folder you want:

```text
skills/ant-plain-language
skills/ant-compact
skills/ant-low-words
```

Copy it into a supported skill location, for example:

```text
.github/skills/ant-plain-language
.github/skills/ant-compact
.github/skills/ant-low-words
.agents/skills/ant-plain-language
.agents/skills/ant-compact
.agents/skills/ant-low-words
~/.copilot/skills/ant-plain-language
~/.copilot/skills/ant-compact
~/.copilot/skills/ant-low-words
~/.agents/skills/ant-plain-language
~/.agents/skills/ant-compact
~/.agents/skills/ant-low-words
```

For a project repository, prefer:

```text
.github/skills/<skill-name>
```

or:

```text
.agents/skills/<skill-name>
```

For personal use across projects, prefer:

```text
~/.copilot/skills/<skill-name>
```

Suggested modes:

- `ant-plain-language` — default understandable answers.
- `ant-compact` — short, no-fluff answers.
- `ant-low-words` — ultra-short token-saving answers.
